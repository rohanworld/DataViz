## Run Locally

Clone the project

```bash
  git clone https://github.com/rohanworld/DataViz
```

Go to the project directory

```bash
  cd dataviz
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python app.py
```
